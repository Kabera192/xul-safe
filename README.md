# XUL-Safe - School Bus Management System

## Repository Structure

| Directory | Description | Technology |
|-----------|-------------|------------|
| `driver-parent-app/` | Mobile app for drivers and parents | Flutter |
| `admin-frontend/` | Web admin dashboard | Angular |
| `backend/` | REST API backend | Spring Boot (Java) |

## Projects

### Driver & Parent App (`driver-parent-app/`)
Flutter mobile application used by bus drivers to manage routes and trips, and by parents to track their children.

### Admin Frontend (`admin-frontend/`)
Angular web application for school administrators to manage buses, routes, conductors, students, and view live tracking.

### Backend (`backend/`)
Spring Boot REST API providing authentication, route management, live tracking, notifications, and reporting for all frontend clients.
